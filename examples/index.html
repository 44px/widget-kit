widget-kit examples index
minimal — minimal example of loader/widget separation. No UI, please see logs in console.
banner — a widget that is located inside page content, adapts to parent's width and controls iframe height (like Embedded Tweets, Share Buttons, Facebook Page).
popup — a small button on the edge of a screen that opens fullscreen popup (like UserEcho).
chat — similar to popup but opens small window that overlaps only part of a page (like Intercom and many other chat widgets).
